# webhdfs-client

## installation
```
pip install git+https://github.com/luff/webhdfs-client.git
```

## configuration
put a .whdfsc.json in your home directory
```
{
  "insecure": false,
  "username": "your-webhdfs-user",
  "password": "your-webhdfs-user-pw",
  "rest_api": "https://your-webhdfs-gateway/webhdfs/v1"
}
```

## usage
```
whdfsc --help
```
